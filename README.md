# shortcite

Short citation styles for Zotero, to be used in posters, slides and abstract submissions.

| style | citation | description |
| --- | --- | --- |
| shortcite.cls | K. Drużbicki *et al*., *Crystal Growth & Design* **24**, 391–404 (2024). | If there are 3 or more authors, name just the first one |
| shortcite-doi.cls | 1. K. Drużbicki *et al*., *Crystal Growth & Design* **24**, 391–404 (2024). doi:10.1021/acs.cgd.3c01112 | Same as shortcite.cls but with the DOI |
| shortcite-3auth-doi.cls | K. Drużbicki, P. Gila-Herranz, P. Marin-Villa *et al*., *Crystal Growth & Design* **24**, 391–404 (2024). doi:10.1021/acs.cgd.3c01112 | If there are more than 4 authors, name just up to the 3rd one; also include DOI |
